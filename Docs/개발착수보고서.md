**2\. 시스템 기능 개발 범위**

(\*착수보고시 보고되지 않은 기능은 임의로 추가하여 개발할 수 없습니다. 최대한 상세히 기술바랍니다.)

**1\) 시온로그인 연동 및 최초 등록:**
\- 시온로그인(고유번호 + 비밀번호)으로 로그인하며, 성도 정보(이름, 부서 등)는 시온로그인에서 자동 수신
\- 최초 로그인 시 시온로그인에 없는 정보(성별, 차량 정보)를 별도 입력 화면(/setup)에서 등록
\- 로그인 후 관리자 수동 승인(PENDING → APPROVED) 완료 전까지 서비스 이용 불가

**2\) 카풀 게시글 등록 및 조회:**
\- 승인된 운전자 회원이 카풀 게시글(출발지, 목적지, 출발 시각, 최대 탑승 인원, 성별 매칭 옵션, 공지) 등록
\- 전체 승인 회원이 메인 피드에서 게시글 카드 목록 조회 및 성별·잔여석 필터 적용
\- 출발 시각은 현재 시각 기준 +10분 이상, +7일 이내로 제한

**3\) 탑승 신청 및 승인/거부:**
\- 탑승자 회원이 게시글에 탑승 신청(탑승 희망 위치 입력)
\- 운전자가 신청 목록 확인 후 개별 승인 또는 거부
\- 성별 매칭 옵션(동성만) 설정 시 이성 신청 자동 차단

**4\) 운행 상태 관리:**
\- 운전자가 [출발] 버튼 클릭 시 게시글 상태 STARTED 전환
\- 출발 시각으로부터 5분 내 [출발] 미클릭 시 게시글 자동 CANCELLED 처리 (스케줄러)
\- 운행 완료 후 [완료] 버튼 클릭 시 COMPLETED 처리 및 탑승자 내역 확정

**5\) 취소 및 노쇼 처리:**
\- 출발 30분 전 이상: 자유 취소
\- 출발 10~30분 전: 취소 기록 남김
\- 출발 10분 이내 취소: 노쇼 처리 (노쇼 횟수 누적)
\- 노쇼 3회 누적 시 30일 이용 정지 자동 처리

**6\) 운전자 등록:**
\- 최초 로그인 셋업 또는 마이페이지에서 차량번호·차종·최대 탑승 인원 등록
\- 자동차 보험 가입 확인, 운전면허 유효 확인, 비영리 목적 확인 3종 체크 필수

**7\) 신고 기능:**
\- 탑승자/운전자 간 노쇼·무례·위험운전·요금 문제·기타 사유로 상호 신고 가능
\- 신고 내역은 관리자 페이지에서 확인 및 처리

**8\) 마이페이지:**
\- 내 프로필(이름, 부서, 성별), 차량 정보, 계좌 정보 조회 및 수정
\- 내 카풀 게시글 이력 및 탑승 신청 이력 조회
\- 계좌번호는 본인만 조회 가능 (복호화 표시)

**9\) 관리자 기능:**
\- 회원 목록 조회, PENDING 회원 승인/거부, 회원 이용 정지 처리
\- 전체 게시글 조회 및 강제 취소
\- 신고 내역 조회 및 처리
\- 전체 회원 및 게시글 이력 엑셀 다운로드(백업)

**3\. 시스템 구성**

 1\) 프론트엔드: Next.js 15 App Router (TypeScript) — 모바일 웹 최적화, GitHub Pages 정적 배포
 2\) 백엔드: Spring Boot 3.3 (Java 17) — REST API 서버, 별도 서버 배포
 3\) 데이터베이스: PostgreSQL — 회원 정보, 카풀 게시글, 신청 내역, 신고 내역 저장
 4\) 시온로그인 연동: 시온로그인 API를 통해 성도 정보(이름, 부서, 연락처) 수신 및 계정 인증

**\< 개발 시 필수 체크 사항 \>**

1\. 개발착수 보고 검토 주체
\- 시온로그인 연동이 필요하므로, 총회 행정서무부와 함께 검토합니다.
\- 담당자 변경 시 총회에 보고해야 합니다.

• 착수 보고 승인 후 개발 프로세스
1\) 착수 보고서 제출 및 승인
2\) 화면설계서 제출 및 승인
3\) 개발 착수
4\) 개발 완료 후 총회에서 최종 통합 테스트 및 보안점검 진행
5\) 총회 인증마크 발급 및 성도 대상 공식 서비스 배포

2\. 개발 진행 시 유의 사항
\- 모든 어플리케이션은 개발자 센터에 등록하여 사용해야 합니다.
\- 본 서비스는 지파/교회 범위에서 사용하는 서비스로, 총회 전산과와 사전 협의하여 진행합니다.

• 시온로그인 연동
성도 정보를 활용하고 계정 관리가 필요한 시스템으로, '시온로그인' 시스템을 연동합니다. 서비스 자체적으로 계정을 운영하거나 성도 정보를 별도로 수집하지 않습니다.

• 총회 기능 중복 개발 금지
총회에서 이미 제공하고 있는 기능과 중복 개발되지 않도록 확인 후 진행합니다.

• 법적 요건 및 품질 보장
개인정보보호법 등 법적 기준을 철저히 준수하며, 분담금은 유류비 분담 성격의 3,000원 정액으로 고정하여 유상운송에 해당하지 않도록 운영합니다. 서비스 이용약관·개인정보처리방침·위치정보 이용약관 3종을 제공합니다.
